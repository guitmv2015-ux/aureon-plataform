# Assets estáticos esperados

Este projeto referencia os seguintes arquivos que devem ser adicionados antes do deploy em produção:

- `logo.png` — usado no schema markup (Organization JSON-LD) e pode ser usado no admin do Payload.
- `og-default.jpg` (1200×630px) — imagem padrão de Open Graph/Twitter Card para páginas sem imagem própria.
- `favicon.ico`, `icon.png` ou `apple-icon.png` — favicons (o Next.js App Router detecta automaticamente arquivos com esses nomes em `src/app/`).

Nenhum desses arquivos foi incluído neste scaffold para evitar a inclusão de imagens placeholder genéricas — substitua por assets de marca reais da Aureon Partners.
